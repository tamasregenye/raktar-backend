## Segédletek

### Használható segédletek

* Jest dokumentáció (https://jestjs.io/docs/using-matchers)
* ```tests``` mappa, ahol a korábi tesztek találhatók
* Swagger dokumentáció (http://localhost:3000/api/docs/)
    * ehhez szükség van a szerver futtatására

### Szerver futtatása

* húzza le a legutóbbi módosításokat (fetch majd pull)
* csomagok telepítése: ```npm i```
* teszt szerver futtatása: ```npm run start-test```

### Tesztesetek futtatása

* az összes teszt futtatása: ```npm run test```
* HTML riport generálása: ```npm run test:html``` scriptet. 
    *  a generált fájl a `test_reports` mappában található
